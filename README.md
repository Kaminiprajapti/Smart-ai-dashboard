# 🚀 Smart AI Sales Dashboard (Code With Gemini)

An advanced analytics dashboard that uses **Google Gemini 1.5 Flash** to provide real-time business insights and predictive forecasting.

## ✨ Key Features
- **AI-Powered Insights:** Uses Gemini API to analyze sales trends and suggest business strategies.
- **Dynamic Forecasting:** Predictive modeling that adapts to growth or downshift trends.
- **Interactive Visualization:** Styled Pandas tables and Matplotlib gradients for better data readability.
- **Custom Filters:** Sidebar controls to filter data by Region and Category.
- **Data Export:** Capability to download filtered reports as CSV.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **AI Model:** Google Gemini API
- **Data Handling:** Pandas, NumPy
- **Visuals:** Matplotlib, Seaborn

## 🚦 Getting Started
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file and add your `GEMINI_API_KEY`.
4. Run the app: `streamlit run app.py`
